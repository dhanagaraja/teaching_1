API Endpoints Documentation

Method Endpoint Description
GET /amzon/api/products/ List all products (with search & price filters)
POST /amzon/api/products/ Create a new product (JSON body)
GET /amzon/api/products/{id}/ Retrieve a specific product
PATCH /amzon/api/products/{id}/ Partial update product (JSON body)
DELETE /amzon/api/products/{id}/ Delete a product
Query Parameters for List:
  • q - Search in name and description
  • min_price - Filter by minimum price
  • max_price - Filter by maximum price

Method Endpoint Description
GET /amzon/api/cities/ List all cities
POST /amzon/api/cities/ Create a new city (JSON body)
GET /amzon/api/cities/{id}/ Retrieve a specific city
PATCH /amzon/api/cities/{id}/ Partial update city (JSON body)
DELETE /amzon/api/cities/{id}/ Delete a city
